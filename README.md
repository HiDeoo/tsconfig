## Usage

> **Note** This configuration requires at least TypeScript v4.4.0.

### Install

```shell
$ pnpm add -D typescript @hideoo/tsconfig
```

### Configure

Reference the configuration in your `tsconfig.json` file:

```json
{
  "extends": "@hideoo/tsconfig"
}
```
